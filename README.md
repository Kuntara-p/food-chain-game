# 🌿 ตะลุยดินแดนห่วงโซ่อาหารสุดคิวท์ (Cute Food Chain Adventure)

เกมการศึกษาเชิงปฏิสัมพันธ์ (Interactive Educational Game) ที่ออกแบบมาเพื่อให้เด็กๆ ได้เรียนรู้เรื่อง **"ห่วงโซ่อาหาร (Food Chain)"** ผ่านการลากวาง (Drag & Drop) สิ่งมีชีวิตต่างๆ ลงในช่องที่ถูกต้องตามลำดับการกินอาหารในธรรมชาติ

## 🎮 วิธีการเล่น (How to Play)
1. **เริ่มการผจญภัย:** กดปุ่มเริ่มเกมเพื่อเข้าสู่ด่านต่างๆ
2. **ลากและวาง:** ใช้เมาส์ (หรือนิ้วสัมผัสบนแท็บเล็ต/มือถือ) ลากการ์ดสัตว์หรือพืชจากกระเป๋าด้านล่าง ไปวางลงในช่องว่างด้านบนให้ถูกต้องตามลำดับห่วงโซ่อาหาร
3. **ใช้ลูกศร:** ลากลูกศร "ส่งพลังงาน" วางคั่นระหว่างสิ่งมีชีวิต เพื่อแสดงทิศทางการถ่ายทอดพลังงาน (ใครถูกกิน)
4. **พลิกการ์ด:** กดคลิกที่การ์ดเบาๆ เพื่อพลิกดูคำใบ้หรือข้อมูลน่ารู้ของสิ่งมีชีวิตนั้นๆ
5. **ตรวจคำตอบ:** เมื่อวางครบทุกช่อง ให้กดปุ่ม "ตรวจคำตอบ" เพื่อรับดาว!

## ✨ ฟีเจอร์เด่น (Features)
- **ระบบคำใบ้อัจฉริยะ (Owl Hint System):** คุณนกฮูกผู้ช่วยจะโผล่มาให้คำใบ้พร้อม **เสียงพากย์ (TTS)** หากผู้เล่นคิดไม่ออกหรือปล่อยทิ้งไว้นานเกิน 10 วินาที
- **สมุดสะสมการ์ด (Encyclopedia):** เมื่อเล่นผ่านแต่ละด่าน สิ่งมีชีวิตที่ปรากฏในด่านนั้นจะถูกปลดล็อกเข้าสู่สมุดสะสมการ์ด เพื่อให้ผู้เล่นกลับมาอ่านเกร็ดความรู้เพิ่มเติมได้
- **ระบบเสียงและแอนิเมชัน:** มีเสียงดนตรีประกอบ เสียงประกอบการลากวาง เอฟเฟกต์พลุ (Confetti) และอนุภาค (Particles) เพิ่มความตื่นเต้น
- **การปรับสัดส่วนหน้าจอ (Aspect Ratio Toggle):** ผู้เล่นสามารถกดปุ่มมุมซ้ายล่าง 📺 เพื่อสลับโหมดหน้าจอให้เป็น 16:9 เหมาะสำหรับการฉายโปรเจกเตอร์หรือหน้าจอทุกรูปแบบ
- **รองรับทุกอุปกรณ์ (Responsive):** สามารถเล่นได้ทั้งบนคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)
- **HTML5:** โครงสร้างของเว็บและหน้าจอเกม
- **CSS3 (Vanilla):** การตกแต่งสไตล์, แอนิเมชัน, Flexbox/Grid และตัวแปรสี (CSS Variables)
- **JavaScript (ES6):** ลอจิกของเกม, Drag & Drop API แบบ Custom (รองรับ Touch), Web Audio API และ SpeechSynthesis API สำหรับเสียงพากย์

## 🚀 การนำไปใช้งาน (Deployment - GitHub Pages)
โปรเจกต์นี้เป็น Static Website ธรรมดา (ไม่มี Backend) สามารถนำขึ้น **GitHub Pages** ได้ฟรีทันที:
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น Repository ของคุณบน GitHub
2. ไปที่เมนู **Settings** ของ Repository
3. เลื่อนลงมาที่หัวข้อ **Pages**
4. ตรงส่วน **Source** ให้เลือกเป็น `main` หรือ `master` branch แล้วกด Save
5. รอประมาณ 1-2 นาที คุณก็จะได้ลิงก์สำหรับให้เด็กๆ เข้ามาเล่นเกมได้ทันที!
